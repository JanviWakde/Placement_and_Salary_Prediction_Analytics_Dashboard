# Placement_and_Salary_Prediction_Analytics_Dashboard

## Objective

The objective of this project is to analyze student placement data and salary trends using an interactive Power BI dashboard. This project transforms raw CSV data into meaningful insights that help understand how academic performance and skill-based factors influence placement outcomes and salary packages.

The dashboard supports data-driven decision-making for improving institutional placement strategies.

---

## Dataset

The project uses a CSV dataset containing the following student attributes:

- CGPA  
- Placement Status (Placed / Not Placed)  
- Salary Prediction
- Techinical Skills
- Number of Hackthons
- CGPA  
- Number of Backlogs  
- Number of Internships  
- Number of Certifications  

The dataset is imported directly into Power BI for analysis and visualization.

---

## Tools & Technologies Used

- **Power BI Desktop** – Dashboard creation and visualization  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – KPI and measure calculations  
- **CSV File** – Data source  

---

## Methodology

### Data Import
- Imported the CSV dataset into Power BI.

### Data Cleaning
- Removed inconsistencies  
- Handled missing values  
- Formatted numerical columns properly  

### Data Modeling
- Created calculated columns and measures using DAX  
- Designed KPIs for performance tracking  

### KPI Development
The following key metrics were created:
- Total Students  
- Placement Rate (%)  
- Average Predicted Salary  
- High Salary Count  

### Dashboard Design
Developed interactive visualizations including:
- CGPA vs Placement Analysis  
- Internship Impact on Placement  
- Backlogs vs Salary Comparison  
- Certifications vs Salary Trends  
- Placement Distribution (Placed vs NotPlaced)  

---

## Key Insights

- Higher CGPA significantly increases placement probability.  
- Internship experience positively impacts employability.  
- Fewer backlogs are associated with higher salary packages.  
- Certifications contribute to improved salary outcomes.  
- Only 42% of students were placed, indicating scope for improvement.
